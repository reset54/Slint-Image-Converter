# Slint Image Converter
Tool for help convert images, update splashes, etc
based on Slint, Rust

## Tech Stack

* **Language:** Rust (Edition 2024)
* **UI Framework:** [Slint](https://slint.dev/)

Getting Started
Prerequisites

    Rust 1.85+ (for Edition 2024 support)

    Cargo

## Build and Run

    1. Clone the repository:
```bash
git clone [https://github.com/reset54/Slint-Image-Converter.git](https://github.com/reset54/Slint-Image-Converter.git)
```
```bash
cd Slint-Image-Converter
```
### 2. Build the project:
```bash
cargo build
```
### 3. Run the application:
```bash
cargo run
```
